# BlogVerse ✨

BlogVerse is a full-stack blog platform built using the MERN stack. Users can create, edit, manage, and interact with blogs through a modern and responsive interface.

## ✨ Features

### 👤 User Features

* User Registration
* User Login
* Logout
* Forgot Password (Email Reset)
* Welcome User Dashboard

### 📝 Blog Features

* Create Blogs
* Edit Blogs
* Delete Blogs
* Search Blogs
* Comment System

### 🖼️ Media Features

* Image Upload
* Video Upload

### 🎨 UI Features

* Responsive Design
* Dark Mode
* Toast Notifications

### 🛡️ Admin Features

* Admin Dashboard
* User Management
* Block/Unblock Users
* Platform Statistics
* Delete Any Blog
